连接mysql失败的问题
mysql

原因之一，可能是iptables问题：

可以进行相关端口开放设置：

sudo iptables -I INPUT -s 127.0.0.1 -p tcp --dport 3306 -j ACCEPT
sudo iptables -I INPUT -s 192.168.1.101 -p tcp --dport 3306 -j ACCEPT
sudo iptables -nL | grep 3306
sudo service iptables save
sudo service iptables restart
